Tesi in Latex
TeX è un sistema per l'impaginazione di testi, progettato per garantire un'ottima qualità estetica del documento finito. LaTeX è una raccolta di macro per TeX che rende più semplice la scrittura di documenti TeX.

E` fortemente consigliato, sebbene non obbligatorio, scrivere la prova finale in LaTeX, in allegato alla pagina potete trovare un frontespizio predefinito ed uno scheletro-tesi di esempio .tex.

Di seguito troverete tutte le informazioni utili per imparare a utilizzare questo potentissimo linguaggio di formattazione.

Lo sviluppo di un documento LaTeX prevede tre fasi:

Stesura del documento LaTeX nomefile.tex con un editor di testo.
Compilazione del file sorgente nomefile.tex.
Visualizzazione o stampa del file nomefile.dvi con un opportuno programma.
